# PokeFavs

- [Preview page](https://pokefavs.netlify.app/)

App para elegir tus pokemones favoritos haciendo uso de la APi de pokemones

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
